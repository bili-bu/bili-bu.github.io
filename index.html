Projects

1. Princess Peach avoids her friends

This was my first project at General Assembly. We were given the task to build a grid-based game. I used HTML and CSS as well as "plain vanilla" JavaScript to create the game. I chose to recreate the cult classic Frogger with a SuperMario theme.

2. The Harry Potter Encyclopedia

This was a hackathon-like pair-programming project to build a frontend React application that consumed a public API. My project partner and I decided to use a public API for Harry Potter to render a full encyclopedia of characters, spells and house information. We also created an interactive section in which a visitor to the website could be placed in a Hogwarts house by means of the sorting hat.

3. The Trivia Game

For the third project we were tasked to build our first full-stack application. We used React for the front-end and Django for the backend. We decided to design this application as a mobile phone application since we wanted our users to feel like they could play anytime and anywhere.

4. Hello Code

For the fourth project, my classmates and I decided to create a chat app which connects coders within a community to help each other with questions on a language or framework. This was a full-stack application using React Hooks for the frontend components, Django for the backend architecture and Material UI for styling.
